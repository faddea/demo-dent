# Estructura del Proyecto - Clínica Odontológica

## Página: Inicio (index.html)

### Secciones:
1. **Navbar** (nav)
   - Logo (dinámico desde config)
   - Links: Servicios, Nosotros, Testimonios
   - Botón "Agendar Turno"
   - Menu mobile (slide from right)

2. **Hero** (section#hero)
   - Título principal
   - Subtítulo/descripción
   - Botones: "Agendar Turno", "Ver Servicios"
   - Imagen (placeholder)

3. **Servicios** (section#servicios)
   - Título "Nuestros Servicios"
   - Grid de 6 servicios (renderizado desde config.json)

4. **Sobre Nosotros** (section#sobre-nosotros)
   - Imagen (placeholder)
   - Título "Cuidamos tu Sonrisa"
   - Descripción
   - Stats: Años de experiencia, Pacientes satisfechos

5. **Testimonios** (section#testimonios)
   - Título
   - Grid de testimonios (renderizado desde config.json)

6. **Contacto/CTA** (section#contacto)
   - Título
   - Formulario: Nombre, Teléfono, Email, Mensaje

7. **Footer** (footer)
   - Logo
   - Contacto (teléfono, email, dirección)
   - Horario
   - Redes sociales

8. **WhatsApp Float**
   - Botón flotante fijo

## Archivos:
- index.html - Estructura
- index.js - Funcionalidad y renderizado dinámico
- styles.css - Estilos custom
- config.json - Datos de la clínica
